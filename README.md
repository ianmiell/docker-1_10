Docker 1.10 Demo
================

An automated, annotated, interactive demo of Docker 1.10's new features.

```
pip install shutit
git clone --recursive https://github.com/ianmiell/shutit-docker-1_10
cd shutit-docker-1_10
./run.sh
```
