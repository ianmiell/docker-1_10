Docker 1.9 Demo
================

An automated, annotated, interactive demo of Docker 1.9's new features.

```
pip install shutit
git clone --recursive https://github.com/ianmiell/shutit-docker-1_9
cd shutit-docker-1_9
./run.sh
```
